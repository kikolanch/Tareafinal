El lenguaje de las etiquetas y el arte del estilo

🧩 Conceptos Básicos de HTML y CSS

Para construir cualquier rincón del mundo digital moderno, necesitamos combinar dos tecnologías obligatorias que trabajan en perfecta sincronía:

La analogía del cuerpo: HTML es el esqueleto (los huesos que dan soporte a la página), mientras que CSS es la ropa, el peinado y el maquillaje (lo que hace que el cuerpo luzca hermoso y presentable).

🧱 Entendiendo el Lenguaje HTML

HTML funciona mediante etiquetas. Las etiquetas son palabras clave rodeadas por los símbolos < y > que le dicen al navegador qué tipo de contenido está leyendo. Aquí tienes el catálogo esencial que hemos aprendido:

Etiqueta HTML ¿Qué es de forma sencilla?
<html>El inicio y fin. Le dice al ordenador: "A partir de aquí comienza una página web".
<head>La mente invisible. Guarda los datos técnicos, el título de la pestaña y los estilos. El usuario no lo ve directamente en la pantalla.
<body>El cuerpo visible. Es el contenedor que guarda todo lo que sí vas a ver: textos, imágenes, botones y menús.
<h1> a <h6>Los títulos. <h1> es el título más grande e importante, y <h6> es el subtítulo más pequeño.
<p>El párrafo. Sirve para escribir bloques de texto normales y corrientes de manera ordenada.
<img>La imagen. Sirve para mostrar fotos, ilustraciones o gráficos en la pantalla indicando su procedencia.
<a>El enlace. Es el botón o texto interactivo que, al pulsarlo, te lleva volando a otra página web o sección.
<ul> y <li>Las listas. <ul> crea una lista con puntos en blanco y <li> define cada uno de los elementos de esa lista.

🎨 Qué es CSS (Cascading Style Sheets)

CSS significa Hojas de Estilo en Cascada. Es el lenguaje encargado exclusivo de la parte estética y visual de la web. Sin CSS, todas las páginas de internet se verían iguales: textos negros aburridos sobre un fondo blanco plano.

Gracias al CSS podemos cambiar los colores de fondo, las tipografías, añadir bordes elegantes y controlar con precisión los espacios mediante dos herramientas fundamentales:

  • Padding (Relleno): El espacio que hay *dentro* de una caja (la distancia entre el texto y su propio borde).
  • Margin (Margen): El espacio que hay *fuera* de una caja (la distancia de separación con los demás elementos vecinos).

Un vistazo a cómo se escribe CSS:

/* Le decimos a todos los títulos h1 que sean marrones y elegantes */
h1 {
    color: #7c5c43; /* Color café */
    font-family: 'Playfair Display'; /* Letra clásica */
    margin-bottom: 20px; /* Espacio hacia abajo */
}
